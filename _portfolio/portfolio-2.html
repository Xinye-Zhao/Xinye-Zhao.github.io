--- title: "C++ Based 3D Pacman Game" excerpt: "Built a 3D Pacman game with OpenCV, DFS-based maze logic, multithreading, and lighting effects." collection: portfolio ---

Project using C++ and OpenCV from September 2021 to December 2022.

  • Developed a 3D maze with OpenCV for a Pacman-style game.
  • Applied DFS for maze traversal and automatic ghost tracking behavior.
  • Used multithreading to enable independent action for each ghost model.
  • Added lighting effects with glLightfv to improve the 3D visual experience.
